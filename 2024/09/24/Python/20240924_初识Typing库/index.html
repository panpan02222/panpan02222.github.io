写在前面

该模块在 Python3.5 版本时引入，归属于 Python 标准库。其主要目的为 : 类型注解.

  • 官方说明 : This module provides runtime support for type hints.

  • 官方举例 :

    该函数目的为：计算立方体的表面积，要求输入立方体的边长 (类型注解 : float), 表示 edge_length 应为一个浮点数。最终函数返回一个字符串类型.(-> str)

使用类型注解可以更好的帮助，我们明确参数和返回值的类型，使得代码更易于理解和维护.
