Linux 快捷指令

发表于 2024-07-17 更新于 2024-09-10 分类于 In Linux 阅读次数： 本文字数： 651 阅读时长 ≈ 2 分钟

本人工作时常用到的一些 linux 快捷指令，希望对大家有所帮助。

  • 查看正在运行的 python 脚本
    ps -ef |grep python

    ps aux | grep python

  • 查看当前目录下的文件占用内存

    du -sh

  • 查看文件系统磁盘占用情况

    df -h $home

  • 计算文件数

    ls | wc -l

  • 查看当前文件夹下每个文件的大小

    ls -lh

  • 暂停进程 18000 秒

    sh -c | sleep 18000

  • 显示物理地址
    overlay

tar

  • 解压 tar.gz file

    tar -zxvf folder.tar.gz folder

  • 解压 tar file

    tar -xvf folder.tar

unrar

  • 解压 rar file

    unrar x example.rar

unzip

  • 解压 zip file

    unzip -l example.zip

远程操作

  • 远程发送文件
    scp folder_path root@ip:folder_path

  • 远程链接服务器
    ssh username@remote_server_ip

挂载

  • 手动挂载 U 盘
  • 批量安装 rpm 包
    rpm -ivh *.rpm

日志

  • 将日志输入到 log 中
    nohup python main.py >log.log 2>&1 &

  • 查看日志
    tail -f log.log

软链接

  • 建立软链接
    ln -s [源文件] [软链接文件]
